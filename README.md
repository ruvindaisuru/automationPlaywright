# UI Automation with Playwright

This repository contains a sample **UI automation project** using **Playwright** with **TypeScript**. The project demonstrates automation testing on a sample e-commerce website.

## Features
- Automated UI testing of the [Sauce Demo](https://www.saucedemo.com/) website.
- Covers **functional test scenarios**, including login, product selection, and checkout workflows.
- Implemented using **Playwright** and **TypeScript**.
- Integrated with **CI/CD pipelines** for automated test execution.
